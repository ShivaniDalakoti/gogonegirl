================================================================================

🚂 STEP 2: Deploy Backend to Railway (5 minutes)

1. Go to https://railway.app
2. Sign up with GitHub
3. Click "New Project" → "Deploy from GitHub"
4. Select "gogonegirl" repo
5. Add Environment Variables:
   PORT=5000
   NODE_ENV=production
   GMAIL_USER=gogonegirl@gmail.com
   GMAIL_PASSWORD=xxxx xxxx xxxx xxxx

6. Click "Deploy"
7. Wait 2-3 minutes
8. Copy your backend URL (looks like: https://gogonegirl-production.railway.app)

================================================================================

🔷 STEP 3: Deploy Frontend to Vercel (5 minutes)

1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "Add New" → "Project"
4. Select "gogonegirl" repo
5. Set Root Directory: app
6. Click "Deploy"
7. Go to Settings → Environment Variables
8. Add: VITE_API_URL = https://gogonegirl-production.railway.app
9. Go to Deployments → Click "Redeploy"
10. Wait 2-3 minutes
11. Copy your frontend URL (looks like: https://gogonegirl.vercel.app)

================================================================================

⚙️  STEP 4: Update CORS (2 minutes)

Edit server.js around line 14, add your Vercel URL:

    app.use(cors({
      origin: [
        'http://localhost:5173',
        'https://gogonegirl.vercel.app',  // ← ADD YOUR VERCEL URL HERE
      ],
      credentials: true
    }));

Then push:
    git add server.js
    git commit -m "Update CORS for production"
    git push origin main

Railway auto-redeploys!
