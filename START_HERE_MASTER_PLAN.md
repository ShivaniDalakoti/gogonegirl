## 📞 NEED HELP?

### If the app won't start:
1. Check that Node.js is installed: `node --version`
2. Check that dependencies are installed: `npm list` (both root + app)
3. Clear cache: `rm -rf node_modules package-lock.json && npm install`
4. Restart everything

### If screen recording is laggy:
1. Close unnecessary apps
2. Record at lower resolution first (1280×720 instead of 4K)
3. Use QuickTime (Mac) or ScreenFlow—they're most stable

### If voiceover doesn't sync:
1. Listen to both demo + voiceover
2. Trim or add silence to make them match
3. Export voiceover as WAV (better quality than MP3)

### If video won't export:
1. Try a different video editor (CapCut is usually more forgiving than iMovie)
2. Reduce resolution to 1080p
3. Check that you have enough disk space
