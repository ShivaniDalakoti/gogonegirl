## 🎥 FILMING INSTRUCTIONS

### Best practices:
1. **Record at 1080p or higher**
2. **Use QuickTime Screen Recording** (Mac):
   - Cmd + Shift + 5
   - Select "Record Selected Portion"
   - Record window size: 1280×720 or 1920×1080
   - Click Start
3. **Keep mouse visible** — Click deliberately, don't hover randomly
4. **No super-fast clicking** — Moves between sections at normal browsing speed
5. **Pause briefly** (1-2 sec) at key moments so viewer can read

### Screen recording flow:
- Start recording on home page (hero section visible)
- Click "Explore Trips"
- Wait for load (pause 1 sec)
- Click on Himalayan Escape
- Wait for load (pause 1 sec)
- Show trip details (scroll slightly)
- Click "Book Now"
- Wait for load (pause 1 sec)
- Show booking form
- STOP recording

**Total recording time: ~1:15 (leave room for voiceover timing)**

## 🚨 IF SOMETHING BREAKS DURING DEMO RECORDING

1. **App doesn't load?** Restart: `npm start` (root) + `npm run dev` (app directory)
2. **Trip page slow?** It might fetch from backend. Give it 5 seconds, then try again.
3. **Booking form doesn't show?** Make sure you're logged in or try /register first.
4. **Screenshot looks pixelated?** Record at higher resolution or reduce browser zoom to 90%

**Don't film the error.** Close browser, restart, and try again. This is for real—you get one shot with investors.
